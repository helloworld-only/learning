##### 2、添加字体高亮快捷设置

>  打开偏好设置 -->> 打开高级设置 -->> 使用记事本打开conf.user.json文件 -->> Ctrl + f 找到 keyBinding
>
>  ```json
>  "keyBinding": {
>      // for example: 
>      // "Always on Top": "Ctrl+Shift+P",
>  
>      "Highlight": "Ctrl+q"            //追加   "高亮": "快捷方式"
>  }
>  ```
